Image Debugger
--------------

Purpose: To assist with printf style debugging during shader development.
Language: c++/cli
Platform: MS Windows

Features:
-Debug multiple images at the same time.
-View images with greater as one byte per channel data (e.g. floating point, int32 etc)
-Easy to use XML format for HDR images/buffers
-Open source, check http://wiki.github.com/ijc123/Image-Debugger

Usage:
Pan the image by dragging with the left mouse button
Right click on a image for a context menu, e.g. modify exposure settings for HDR images
Mousewheel up/down to zoom in/out or use the middle mouse button
